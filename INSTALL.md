# 安装指南

## 🚀 快速安装

### 第一步:准备文件
确保你已经下载了完整的扩展文件夹,包含以下文件:
```
Chrome_web_page_QRcode/
├── manifest.json
├── popup.html
├── popup.js
├── popup.css
├── lib/
│   └── qrcode.min.js
├── icons/
│   ├── icon16.png
│   ├── icon48.png
│   └── icon128.png
└── README.md
```

### 第二步:打开Chrome扩展管理页面
1. 打开Chrome浏览器
2. 在地址栏输入: `chrome://extensions/`
3. 按回车键

### 第三步:开启开发者模式
在扩展管理页面的**右上角**,找到并打开"开发者模式"开关。

### 第四步:加载扩展
1. 点击页面左上角的"**加载已解压的扩展程序**"按钮
2. 在弹出的文件选择窗口中,找到并选择 `Chrome_web_page_QRcode` 文件夹
3. 点击"选择"按钮

### 第五步:完成!
🎉 恭喜!扩展已成功安装。你应该能在Chrome工具栏看到"网页二维码"图标。

## 📖 使用方法

### 基本使用
1. 打开任意网页(例如: www.baidu.com)
2. 点击浏览器工具栏中的"网页二维码"图标(蓝色图标)
3. 稍等片刻,二维码会自动生成
4. 使用手机扫描二维码即可在手机上打开该网页

### 下载二维码
- 点击弹出窗口中的"下载二维码"按钮
- 二维码图片会保存到你的下载文件夹
- 文件名格式: `qrcode-网站域名.png`

## ⚠️ 注意事项

### 无法使用的页面
以下类型的页面无法生成二维码:
- Chrome内部页面(如 `chrome://extensions/`)
- Chrome Web Store页面
- 浏览器设置页面
- 本地文件页面(如 `file:///`)

### 为什么?
这是Chrome的安全限制,扩展无法访问这些特殊页面的信息。

### 解决方法
只需要切换到普通的网页(如百度、Google等)即可正常使用。

## 🔧 故障排除

### 问题1: 图标不显示
**解决方法:**
- 刷新扩展管理页面
- 重新加载扩展
- 重启Chrome浏览器

### 问题2: 二维码显示"⚠️ 加载失败"
**解决方法:**
- 检查网络连接
- 刷新当前页面
- 重新点击扩展图标

### 问题3: 二维码中间没有显示favicon
**原因:** 某些网站没有favicon或favicon加载失败
**说明:** 这是正常的,不影响二维码的功能,扫码后依然可以正常访问网页

### 问题4: 无法下载二维码
**解决方法:**
- 检查Chrome的下载权限
- 检查下载文件夹是否有写入权限
- 尝试右键点击二维码 → 另存为

## 🎨 自定义说明

### 修改二维码大小
打开 `popup.js`,找到 `generateQRCode` 函数中的:
```javascript
width: 300,
height: 300,
```
修改数值即可(建议保持正方形)。

### 修改颜色主题
打开 `popup.css`,找到 `:root` 部分,修改颜色变量:
```css
--primary-color: #007AFF;  /* 修改主色调 */
--text-primary: #1d1d1f;   /* 修改文字颜色 */
```

### 修改窗口大小
打开 `popup.css`,找到 `body` 部分:
```css
width: 380px;
min-height: 520px;
```

## 🌟 小技巧

### 技巧1: 固定到工具栏
右键点击扩展图标 → 选择"固定",图标会一直显示在工具栏上。

### 技巧2: 设置快捷键
1. 访问 `chrome://extensions/shortcuts`
2. 找到"网页二维码"
3. 设置你喜欢的快捷键(如 `Ctrl+Shift+Q`)

### 技巧3: 批量处理
打开多个标签页,依次点击扩展图标,快速生成多个二维码。
