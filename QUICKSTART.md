# 快速开始指南 ⚡

## 5分钟快速上手

### 步骤1: 加载扩展 (2分钟)
1. 打开Chrome浏览器
2. 地址栏输入: `chrome://extensions/`
3. 打开右上角"开发者模式"开关
4. 点击"加载已解压的扩展程序"
5. 选择 `Chrome_web_page_QRcode` 文件夹

### 步骤2: 测试功能 (1分钟)
1. 打开项目中的 `test.html` 文件
2. 点击浏览器工具栏的扩展图标(蓝色)
3. 查看生成的二维码

### 步骤3: 实际使用 (2分钟)
1. 访问任意网站(如 www.baidu.com)
2. 点击扩展图标
3. 用手机扫描二维码
4. 在手机上打开该网页

## 🎯 核心功能

### ✅ 自动生成二维码
- 点击即用,无需任何配置
- 支持所有普通网页
- 自动获取网站favicon

### ✅ 精美的视觉设计
- 苹果风格的现代UI
- 流畅的动画效果
- 响应式布局

### ✅ 一键下载
- 点击"下载二维码"按钮
- 自动保存为PNG图片
- 文件名包含网站域名

## 💡 使用技巧

### 固定扩展图标
右键点击扩展图标 → 选择"固定"

### 设置快捷键
1. 访问 `chrome://extensions/shortcuts`
2. 为"网页二维码"设置快捷键
3. 推荐: `Ctrl+Shift+Q` 或 `Alt+Q`

### 保存二维码
方法1: 点击"下载二维码"按钮
方法2: 右键点击二维码 → 另存为图片

## 🔧 常见问题

### Q1: 看不到扩展图标?
**A:** 检查扩展是否已启用,刷新页面后重试

### Q2: 提示"无法生成二维码"?
**A:** 检查是否在Chrome内部页面,切换到普通网页即可

### Q3: 二维码中间没有图标?
**A:** 某些网站没有favicon,这不影响二维码功能

### Q4: 扫码后打不开?
**A:** 确保手机和电脑在同一网络,或检查URL是否正确

## 🌟 高级功能

### 自定义二维码尺寸
打开 `popup.js`,修改第70行的数值:
```javascript
canvas.width = 300;  // 修改为你想要的尺寸
canvas.height = 300;
```

### 修改主题颜色
打开 `popup.css`,修改第14行:
```css
--primary-color: #007AFF;  /* 改成你喜欢的颜色 */
```

### 调整窗口大小
打开 `popup.css`,修改第56-57行:
```css
width: 380px;      /* 修改宽度 */
min-height: 520px; /* 修改高度 */
```
